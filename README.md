# Image-Processing
Object Oriented Image Manipulator built with C++. <br>
Based on the graduate level Unix/C,C++ course at the University of Oregon


## Functionalities
Image Shrinking <br>
Rotation <br>
Blending <br>
Subtraction <br>
Bluring <br>
Concatenation (Left-Right, Top-Bottom) <br>
Mirroring <br>
GrayScale conversion <br>
Coloring <br>
